## 可信度分级量尺

- **[Confirmed]**: 基于 30+ 样本的量化计算结论 → 属于确定性规律，可直接作为策略依据
- **[Indicative]**: 基于 10-30 样本的趋势性指向 → 属于强相关趋势，具有高度参考价值
- **[Speculative]**: 基于有限样本或外部推断 → 属于前瞻性洞察，需在实操中动态验证
