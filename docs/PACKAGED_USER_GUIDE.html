XHS 博主分析助手使用说明

这份说明写给完全不懂技术的用户。你只需要会打开浏览器、解压文件、双击程序，就可以分析一个小红书博主主页最近约 100 篇公开内容。

先记住一句话：不能只安装浏览器插件。请先双击 XHS分析助手.exe 启动本地工具，再安装插件并分析博主。
1. 解压不要直接在压缩包里双击文件，先完整解压。
2. 启动双击 XHS分析助手.exe，等待本地页面打开。
3. 分析登录小红书，打开博主主页，点插件里的“分析当前博主”。

第一次使用

  1. 把下载到的 zip 压缩包解压到桌面或其他容易找到的位置。
  2. 打开解压后的文件夹，双击 XHS分析助手.exe。
  3. 如果出现黑色窗口，请不要关闭。工具运行期间它需要保持打开。
  4. 浏览器正常会自动打开 http://127.0.0.1:8000。

如果 Windows 提示“未知发布者”，这是测试版没有代码签名导致的。只从作者提供的正式下载链接获取压缩包。

安装浏览器插件

Chrome

  1. 地址栏输入 chrome://extensions/。
  2. 打开右上角“开发者模式”。
  3. 点击“加载已解压的扩展程序”。
  4. 选择解压文件夹里的 browser-extension。

Edge

  1. 地址栏输入 edge://extensions/。
  2. 打开“开发人员模式”。
  3. 点击“加载解压缩的扩展”。
  4. 选择解压文件夹里的 browser-extension。

看到类似下面的卡片，说明插件已经安装成功，并且是启用状态。

接着可以把插件固定到浏览器右上角，之后点击图标就能使用。

分析一个博主

  1. 确认 XHS分析助手.exe 正在运行，本地网页能打开。
  2. 在 Chrome 或 Edge 里登录小红书。
  3. 打开你想分析的博主主页，地址通常像 https://www.xiaohongshu.com/user/profile/用户ID。
  4. 点击浏览器右上角插件图标。
  5. 点击“分析当前博主”，等待报告页面自动打开。

分析完成后，你会看到类似这样的报告。

常见问题

插件提示“本地工具没启动”

回到工具文件夹，双击 XHS分析助手.exe。等网页打开后，再点插件分析。

插件提示“先打开博主主页”

说明当前页面不是小红书博主主页。请先打开某个博主主页，再点插件。

插件提示“还没有检测到登录”

请先在当前浏览器登录小红书，然后刷新博主主页，再点插件分析。

分析一直没有结束

先不要连续点很多次。等 1-3 分钟，看本地网页里的任务进度。如果失败，请把页面截图和 logs 文件夹发给作者排查。

隐私和使用边界

  • 插件只把必要信息发送到你自己电脑上的 127.0.0.1。
  • 不需要手动复制 Cookie，不需要提供小红书密码。
  • 测试版可能询问是否允许匿名改进；未明确允许时不会发送远端事件。
  • 匿名改进只记录启动、分析状态、报告打开和主动反馈，不包含 Cookie、账号链接、笔记或报告内容，并可在设置中关闭。
  • 不要频繁批量分析大量账号。
  • 报告是基于公开主页样本的观察，不是平台官方数据或保证结论。

关闭和卸载

  1. 关闭本地工具：双击 关闭本地工具.bat。
  2. 删除插件：打开 chrome://extensions/ 或 edge://extensions/，找到插件后点击“移除”。
  3. 删除本地文件：不想用了，可以删除整个工具文件夹。

测试版建议一次只分析一个账号。遇到问题时，请提供截图、电脑系统版本、浏览器类型，以及工具文件夹里的 logs 文件夹。
