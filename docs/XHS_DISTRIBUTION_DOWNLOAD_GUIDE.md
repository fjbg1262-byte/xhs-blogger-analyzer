# XHS 博主分析助手：下载、安装和使用说明

这份说明可以放进小红书虚拟商品交付的 PDF 里，也可以单独发给内测用户。  
请先完整看一遍，再开始操作。

> 当前是 Windows 测试版，不是小红书官方工具。请只用于公开主页内容的学习、复盘和选题参考。

## 一句话说明

XHS 博主分析助手可以帮你分析一个小红书博主主页最近约 100 篇公开内容，并生成一份本地分析报告。

它适合：

- 新手博主研究对标账号
- 小运营团队做内容复盘
- 想快速看清一个账号内容主线的人

它不适合：

- 批量高频监控大量账号
- 代替平台官方数据
- 保证爆款、涨粉或商业结果

## 第 1 步：解压

下载完成后，先把 zip 解压到一个容易找到的位置，例如桌面。

推荐解压后文件夹类似这样：

```text
桌面/XHS博主分析助手-Windows测试版/
```

不要直接在压缩包里面双击文件。一定要先解压。

如果你不会解压，可以这样做：

1. 找到 `XHS博主分析助手-Windows测试版.zip`。
2. 鼠标右键点击它。
3. 选择“全部解压缩”或“解压到当前文件夹”。
4. 等它生成一个新的文件夹。
5. 打开这个新的文件夹。

解压后的文件夹里应该能看到：

```text
XHS分析助手.exe
关闭本地工具.bat
使用说明.html
README-先看我.txt
browser-extension/
docs/
```

## 第 2 步：打开本地工具

双击：

```text
XHS分析助手.exe
```

第一次启动可能会慢一点，这是正常的。  
如果弹出黑色窗口，请不要关闭。工具运行期间需要让它保持打开。

正常情况下，浏览器会自动打开：

```text
http://127.0.0.1:8000
```

如果没有自动打开，可以手动复制这个地址到浏览器地址栏。

## 第 3 步：安装浏览器插件

插件只需要安装一次。

### Chrome 安装

1. 打开 Chrome。
2. 地址栏输入：

```text
chrome://extensions/
```

3. 打开右上角“开发者模式”。
4. 点击“加载已解压的扩展程序”。
5. 选择解压文件夹里的：

```text
browser-extension
```

### Edge 安装

1. 打开 Edge。
2. 地址栏输入：

```text
edge://extensions/
```

3. 打开“开发人员模式”。
4. 点击“加载解压缩的扩展”。
5. 选择解压文件夹里的：

```text
browser-extension
```

## 第 4 步：分析一个博主

1. 确认 `XHS分析助手.exe` 正在运行。
2. 在 Chrome 或 Edge 里登录小红书。
3. 打开你想分析的博主主页。
4. 点击浏览器右上角的插件图标。
5. 点击“分析当前博主”。
6. 等报告页面自动打开。

博主主页通常长这样：

```text
https://www.xiaohongshu.com/user/profile/用户ID
```

## 常见问题

### 插件提示“本地工具没启动”

说明你还没有双击 `XHS分析助手.exe`，或者本地服务没有正常启动。

解决方法：

1. 回到工具文件夹。
2. 双击 `XHS分析助手.exe`。
3. 等网页打开后，再点插件分析。

### 插件提示“先打开博主主页”

说明你当前打开的不是小红书博主主页。

请先打开某个博主主页，再点插件。

### 插件提示“还没有检测到登录”

说明你还没有在当前浏览器登录小红书。

解决方法：

1. 打开 `https://www.xiaohongshu.com/`。
2. 登录你自己的小红书账号。
3. 刷新博主主页。
4. 再点插件分析。

### Windows 提示未知发布者

当前是测试版，没有代码签名，所以 Windows 可能会提示未知发布者。  
请只从作者提供的正式下载包获取文件，不要从陌生来源下载。

### 分析一直没有结束

先不要连续点很多次。

建议：

1. 等 1-3 分钟。
2. 看本地网页里的任务进度。
3. 如果失败，把截图和 `logs` 文件夹发给作者排查。

## 隐私说明

这个工具是本地优先：

- 插件只把必要信息发送到你自己电脑上的 `127.0.0.1`
- 不需要你手动复制 Cookie
- 不需要你提供小红书密码
- 不会默认上传数据到作者服务器

使用建议：

- 一次只分析一个账号
- 不要频繁批量分析大量账号
- 报告只作为样本观察，不要说成平台官方数据

## 关闭和卸载

关闭本地工具：

```text
关闭本地工具.bat
```

删除插件：

1. 打开 `chrome://extensions/` 或 `edge://extensions/`。
2. 找到 `XHS Analyzer Helper`。
3. 点击“移除”。

删除本地文件：

如果不想用了，可以删除整个工具文件夹。
