XHS 博主分析助手 Windows 测试版

第一次使用请按这个顺序：

1. 先完整解压整个压缩包，不要在压缩包里面直接双击。
2. 双击 XHS分析助手.exe。
3. 等浏览器自动打开 http://127.0.0.1:8000。
4. 打开 使用说明.html，按图安装浏览器插件。
5. 在 Chrome 或 Edge 里登录小红书。
6. 打开要分析的博主主页，点击插件里的“分析当前博主”。

不要删除 _internal、runtime、spider_xhs、frontend、browser-extension 等文件夹。
如果插件提示“本地工具没启动”，请先确认 XHS分析助手.exe 正在运行。
遇到问题时，把页面截图、电脑系统版本、浏览器类型，以及 logs 文件夹发给作者排查。
